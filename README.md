# The Stacks Library

A digital library web app for discovering, organizing, and exploring books: your personal stack of reads, all in one place.

## 📚 About

The Stacks Library is a modern web application designed for book enthusiasts who want to manage their reading collections effortlessly. Whether you're tracking books you want to read, currently reading, or have already finished, this app provides an intuitive platform to organize, discover, and explore your literary world.

## ✨ Features

- **Personal Book Collection** - Build and manage your personal library of books
- **Book Discovery** - Explore new books and find recommendations
- **Organize Your Stack** - Categorize and organize books into different stacks or shelves
- **Book Details** - View comprehensive information about each book in your collection
- **Search & Filter** - Easily find books in your collection
- **Responsive Design** - Seamless experience across desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend:** JavaScript (62.9%)
- **Styling:** CSS (21.2%)
- **Markup:** HTML (15.9%)

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (if running a development server)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Jean-Youakim/The-Stacks-Library.git
cd The-Stacks-Library
```

2. Open the application:
```bash
# Option 1: Open index.html directly in your browser
open index.html

# Option 2: Use a local development server (if available)
npm install
npm start
```

## 📖 Usage

1. **Add Books** - Start by adding books to your personal library
2. **Organize** - Create stacks or categories to organize your books
3. **Browse** - Explore your collection and discover new reads
4. **Track Progress** - Keep track of which books you're currently reading
